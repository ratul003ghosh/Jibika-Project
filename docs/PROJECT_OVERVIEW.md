# Jibika Project Overview

Jibika is a PHP and MySQL DBMS lab project for area-wise unemployment tracking and skill-based job matching in Bangladesh.

## Problem

Many job portals only list jobs. They do not clearly show where unemployment is high, which skills are available in an area, which skills employers demand, and where training or NGO/government support is needed.

## Solution

Jibika connects job seekers, employers, admins, and public service stakeholders through:

- Job seeker and employer registration
- Area-wise profiles using district, upazila, and ward
- Employment status tracking
- Job posting and applications
- Skill supply and skill demand analysis
- Admin dashboard analytics
- Reports for unemployment, skills, job demand, and employer activity

## Main Users

- Job seekers: create profile, add skills, search/apply/save jobs, track applications
- Employers: create company profile, post jobs, review applicants, schedule interviews
- Admins: monitor users, jobs, applications, unemployment status, reports, and system activity
- Government/NGO/teachers: understand area-wise unemployment and training needs

## Jibika-Specific Value

The project is not just a job portal. It provides decision support for Bangladesh by showing which areas have more unemployed users, which skills are missing, where jobs are available, and where training programs should be prioritized.
