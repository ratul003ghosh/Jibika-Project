# Reports And Area Monitor

Jibika is designed to help understand unemployment as an area-wise social and economic problem, not only as individual job search.

## Area Monitor Metrics

- District/upazila/ward filter
- Total job seekers
- Unemployed users
- Employed users
- Training users
- Self-employed users
- Jobs available in the area
- Applications submitted in the area
- Skill supply
- Skill demand
- Skill gap

## Reports

The reports page and database views support:

- Area-wise unemployment report
- Skill distribution report
- Job demand report
- Skill gap report
- Training recommendation report
- Employer activity report
- Application status report
- Employment status change report

## Government/NGO Recommendations

High unemployment plus low job availability suggests a need for employment programs.

High demand for a skill but low supply suggests a training program opportunity.

High applications but low hiring can suggest employer-job seeker mismatch, interview bottlenecks, or training needs.
